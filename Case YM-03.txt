# Load the dependencieslibrary(mice)# Impute the missing data m timesimp <- mice(nhanes, m = 5)# Analize completed datasets using linear modelfit <- with(imp, lm(chl ~ bmi + age))# Combine parameter estimatesest <- pool(fit)# Print summary of estimationsummary(est)Пакет MICE використовується для заповнення завантажених значень у даних. Пропущені дані виконуються не один, а кілька разів. Після цього кожен із отриманих наборів навчається на певній моделі. Потім результати агрегуються і видаються підсумкові параметри моделі.Стандартний процес роботи виглядає так як наведено вище.Прізвища команди: Мішкур, Бобошко, Соловйов, Казанцев, Шайтуро з групи УМ-03
